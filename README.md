# 🎵 Mood Swings

Mood Swings is a fun little web app I built for picking music based on how you're feeling. Just choose a mood — Happy, Sad, Romantic, Motivated, or Energetic — and it opens a matching YouTube playlist for you.

🔗 *Live Demo:* [mood-swings-alpha.vercel.app](https://mood-swings-alpha.vercel.app)

## Features
- 🎭 Five mood-based playlists, opening directly in YouTube
- 🔍 Search bar to look up any song on YouTube
- 🎯 A short quiz to help you pick a mood when you're not sure
- ⭐ Active mood highlighting and a "Now Playing" indicator
- 📱 Fully responsive design for mobile and desktop
- 🎨 Custom color themes for each mood

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (no frameworks)
